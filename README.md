# -Android-Unithon-Habitup
<H1>유니톤 Habit-up (습관을 키워주는 어플)<H1>
<H3>해빗업은 개인 습관 별로 기록 및 저장 수정 삭제가 가능합니다
 <H3><br>달성시 캐릭터가 진화 하여 자신만의 캐릭터를 키우고 보관이 가능합니다
 <br> 또한 음성인식하여 캐릭터와 소통이 가능합니다.
 </H3>
